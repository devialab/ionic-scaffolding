# ionic-scaffolding

Very very naive ionic scaffolding


# Installation

* Clone this project
* Copy `scaffolding/` content to wherever you want
* Replace `app-name` with your project name (prefer [kebab-case](http://c2.com/cgi/wiki?KebabCase))
* follow its `README.md` instructions
